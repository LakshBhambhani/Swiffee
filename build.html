A simple Quadruped controlled by your phone(IOS and Android). Runs on Raspberry Pi and Android.

Required Components:

  1. A Phone (Optional)
  2. Raspberry Pi
  3. Xbox Controller (Optional)
  4. 3D Printed parts
  5. 8 9g Servos

Complete Code:

All stl files: https://www.thingiverse.com/thing:3480616

Step 6: Software

Install the following software on your Pi:

  1. Start by installing Debian on Pi
  2. Download Raspbian.
  3. Unzip the file
  4. Write the disc image to your microSD card
  5. Put the microSD card in your Pi and boot up

Step 7: Cloning Repo and Starting server

  1. Open a new terminal on your Raspberry Pi and execute the following:

sudo apt-get update

sudo apt-get upgrade

git clone  

This clones the repo on your Pi.

  1.  Now change directory to cloned repo/Server. Type in the following in the same terminal:

cd Swiffee-Minikame-Simulator

cd Server

  1. To start the server, execute the following in the same terminal:

sudo python3 server.py

This starts the server and now you can run the quadruped using a web browser (Go to <IP Address> Eg: http://192.168.0.28) or you can use the iPhone control app to control it once you change the IP in the app or you can control it using an Xbox Controller when connected to the laptop and the Simulator is running.

Step 8: Controlling it

Method 1: Using a web browser. Head to http://<IP Address of Bot> on a web browser and you can see the various different options you can carry out on the bot. This way to control it is not recommended though.

Method 2: Control using the app for iPhones. You can change the IP Address in the IP box and control it directly using options in the app. Do know that the simulator and Web Browser have way more options than the app. The app is still very handy and way easier than others.

Method 3: Control using an Xbox Controller which is connected to a laptop which is running Swiffee Minikame Simulator on it.
