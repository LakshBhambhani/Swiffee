Swiffee

Swiffee is a smart quadruped (four-legged) robot that can be controlled using an xbox controller or a phone or a laptop or it can run automatically using an on-board vision system

What Can It Do?

Name it and it does it. It can walk in all directions, turn left and right, bow, bend back, stand up, flip back and climb stairs which is coming soon

Who built this?

JavierIH was the primary developer and designer of Minikame. Many different people released remixes of Minikame. Laksh Bhambhani and Megan Choy were one of those groups too. We got a simulator for Minikame, now calling it Swiffee and we got different control methods for the Quadruped
